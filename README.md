How to run the application :-

Download the clone.
Please have nodejs and mongodb installed locally on your machine.
Open cmd and go to the directory having app.js.
Run npm start. ( this will install the required npm modules).
Setup mongodb on your system.
Open new terminal and go to C:/mongodb/bin (or where your mongodb is installed).
Run mongod. (This will start mongodb server).
After mongodb server is running, Go to previous terminal and in the same directory having app.js, run node app.js.
This will start the server locally.
Go to browser at ://localhost:3000.
